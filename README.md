# SvelteSudoku
Svelte를 이용하여 만든 3*3 스도쿠 앱.   
[앱 사용해보기](https://kuman514.github.io/SvelteSudoku/)

# How to play (게임 방법)
- 무작위로 생성되고 32개의 구멍으로 이루어진 스도쿠 판을 1~9의 숫자로 채워넣습니다
- 가로 한 줄과 세로 한 줄 그리고 3*3칸에서 1~9의 숫자는 각 한 번씩만 들어가야 합니다
